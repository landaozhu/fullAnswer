## 对于 fullAnswer 的设想

### 第一期：

在 github 用 readme 的文档方式。
做成一个最全前端面试刷题网站，吸收广大博主的面试总结，吸收各种面经，把答案写透

### 第二期：

- 智能刷题，有点借鉴百词斩，给你展示一道题，然后你自己回答，给自己评分
  系统根据评分给你出题，假如说一道题你自认为一百分，那么不会再给你，如果你只能回答 30 分，就会重复考你。因为经费有限，所以只会做单机版。
- 帮助候选人更好的复习，因为考你才能加深印象
