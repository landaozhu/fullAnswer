## 对于fullAnswer的设想
第一期：
登陆注册、搜索功能（每个问题只能出现一次，不能提出一模一样的问题）、分享面经功能（面经以题目显示，每个题目跟着一个问答链接，这样好复习，因为我不想让我的用户复习的时候再去搜）、收藏功能（希望收藏页面就是一个错题集的形式，考前看一看自己的收藏就好了）、有没有可能在线代码演示
做成一个权威面经网站，前端面经number1

第二期：
智能推荐复习错题，可以让用户手写，然后评分，使用艾什么鬼那个曲线
帮助候选人
错题回顾
第三期：
每个薪资对应面试题
